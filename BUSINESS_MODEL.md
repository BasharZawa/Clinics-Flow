## 3. نموذج التسعير (Pricing Model)

### خطط الاشتراك (SaaS Subscription)

#### الخطة الأساسية (Basic) - $29/شهر
**للعيادات الصغيرة (1-2 أطباء)**
- حتى 100 موعد/شهر
- حتى 500 مريض
- تذكيرات واتساب أساسية
- دعم عبر البريد
- تقارير أساسية

#### الخطة الاحترافية (Pro) - $79/شهر ⭐ الأكثر مبيعاً
**للعيادات المتوسطة (3-5 أطباء)**
- مواعيد غير محدودة
- مرضى غير محدودين
- تذكيرات واتساب متقدمة (24h, 1h)
- إدارة قائمة انتظار
- تقارير متقدمة + Dashboard
- دعم فني عبر الهاتف
- 3 مستخدمين

#### الخطة المتقدمة (Enterprise) - $199/شهر
**للمراكز الكبيرة والشبكات**
- كل مميزات Pro
- مستخدمين غير محدودين
- أكثر من فرع
- API للتكامل
- دعم فني مخصص 24/7
- تخصيص كامل

#### الخطة المخصصة (Custom)
- للشبكات الكبيرة (10+ فروع)
- تسعير حسب المتطلبات
- $999+ /شهر

### رسوم إضافية

| الخدمة | السعر | ملاحظات |
|--------|-------|---------|
| Onboarding وإعداد | $99-299 | لمرة واحدة |
| تدريب فريق | $50/ساعة | عن بعد |
| تخصيصات خاصة | $100/ساعة | تطوير مخصص |
| رسائل واتساب إضافية | $0.01/رسالة | بعد تجاوز الحد |
| النسخ الاحتياطي اليومي | $10/شهر | اختياري |
